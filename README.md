Record-Breaking Lasers Website

How to run locally:
1. Keep index.html and data.csv in the same folder.
2. Open a terminal in that folder.
3. Run: python3 -m http.server
4. Open: http://127.0.0.1:8000

How to update the data:
1. Edit the spreadsheet.
2. Export/download it as CSV.
3. Rename the export to data.csv.
4. Replace the old data.csv in this folder.
5. Refresh the browser.

You can also click Import CSV or drag a CSV onto the page to preview a new file without replacing data.csv.
